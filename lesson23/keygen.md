Тепер імпортуємо всі змінні і функції із новоствореного скрипта:

```bash
source ./keygen.sh
```

Після цього ініціалізуємо середовище:

```bash
key-init
```

Створимо ключі і сертифікати для сервера:

```bash
key-server server1
```

Створимо ключі і сертифікати для клієнта:

```bash
key-client client1
```
